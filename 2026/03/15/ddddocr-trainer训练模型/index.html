ddddocr-trainer训练模型
2026-03-15

下载压缩包并解压

https://github.com/sml2h3/dddd_trainer

cmd进入 ddddocr-trainer

安装Miniconda虚拟环境

https://docs.conda.io/projects/miniconda/en/latest/

创建虚拟环境

conda create -n 名字 python=3.10

启动虚拟环境

conda activate dd

把ddddocr-trainer安装列表里的工具下载了

pip install -r requirements.txt

安装自己显卡对应的torch版本

https://pytorch.org/get-started/previous-versions/

官网寻找对应版本，使用nvidia-smi环境内确认显卡版本

在官网使用全局搜索，没有对应版本就往下降一点，比如我13.1就降为了13.0
然后复制命令，下载对应版本

创建项目

python app.py create 项目名称

准备好数据集

数据集单图片名称为名称_随机hash值

创建缓存

python app.py cache 项目名称 D:\数据集地址

开始训练

python app.py train 项目名称
